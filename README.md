# CineSpot

**CineSpot** is a modern and visually engaging movie discovery app built with **React Native (Expo)**.  
It allows users to explore trending, top-rated, upcoming, and popular movies while viewing detailed overviews, trailers, and personalized favorites â€” all in a seamless, cinematic experience.

---

## Features
- **Browse by Category:** Explore Trending, Popular, Top Rated, and Upcoming movies.
- **Search Functionality:** Find movies by title or genre with instant results.
- **Movie Details:** View synopsis, ratings, release date, genres, and trailers.
- **Recommended Movies:** Get AI-powered recommendations similar to the one youâ€™re viewing.
- **Favorites Section:** Save and manage your favorite movies locally.
- **Context Management:** Handle app-wide state efficiently using React Context.
- **Smooth Navigation:** Optimized stack & tab navigation for fluid transitions.
- **Clean UI:** Consistent and minimal design for an immersive browsing experience.

---

## Tech Stack
- React Native (Expo)
- React Navigation
- TMDB API (for movie data)
- YouTube Data API (for trailers)
- AsyncStorage
- Context API / State Hooks
- Styled Components (for styling)
